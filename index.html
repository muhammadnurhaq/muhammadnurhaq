*Perbedaan Malware dan Ransomware* 1. *Tujuan*: Malware dirancang untuk melakukan tindakan jahat pada sistem komputer atau perangkat lainnya, seperti mencuri data, mengganggu kinerja, atau menyebabkan kerusakan. Ransomware, di sisi lain, dirancang untuk mengenkripsi file atau data dan meminta tebusan untuk membuka kunci. 2. *Jenis*: Malware mencakup berbagai jenis, seperti Virus, Worm, Trojan, Spyware, dan Adware. Ransomware adalah jenis malware yang spesifik. 3. *Dampak*: Malware dapat menyebabkan kerusakan pada sistem, kehilangan data, atau gangguan kinerja. Ransomware dapat menyebabkan kehilangan akses ke file atau data penting, serta biaya tebusan yang tinggi.
